项目结构搭建，路径中不能有中文，否则sqlite找不到文件，而且css加载无效
application()
{
    manage.py               <-脚本数据
    runserver.py            <-启动服务器
    {
        application()       <-web目录
        {
            app.conf        <-配置文件
            models.py       <-数据模型
            views.py        <-视图，url的路径
            __init__.py     <-模块导出文件
        }
        static              <-静态文件
        templates()         <-页面模版
        {
            base.html
            index.html
            login.html
        }
    }
}

filter指定列名的时候，可以不使用类名；
filter_by也不是说任何时候都可以不指定类名
主要就是当使用filter的时候条件之间是使用“=="，fitler_by使用的是"="。
user1 = session.query(User).filter_by(id=1).first()
